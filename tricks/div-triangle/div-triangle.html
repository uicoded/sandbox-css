Code Examples

1. Border Method (Most Common)

.triangle { width: 0; height: 0; border-left: 50px solid transparent; border-right: 50px solid transparent; border-bottom: 80px solid #e74c3c; }

2. CSS Clip-Path Method

.triangle { width: 100px; height: 80px; background: #3498db; clip-path: polygon(50% 0%, 0% 100%, 100% 100%); }

3. Right Triangle

.triangle-right { width: 0; height: 0; border-top: 80px solid #9b59b6; border-right: 80px solid transparent; }
