sRGB colors with the color() function

color(colorspace c1 c2 c3[ / A])

The CSS color() function lets you choose a color in a specific color space. The first argument is the color space to use, which defines the options for the following channels. Like rgb(), you can set the alpha channel by setting a number between 0 and 1, or a percentage, after a /.

A color specified with the rgb() function can also be specified with the same percentage channel values using color(srgb), although the values go from 0 to 1, rather than 0 to 100%. The alpha channel seems to be supported with '/' (4th value did not work)

  • rgb(183 21 64)
  • rgb(72% 8% 25%)
  • color(srgb 0.72 .08 .25)
  • color(srgb 0.72 .08 .25 / 0.5)
