Nth-Child Automatic Positioning

No custom box-[number] classes needed - uses automatic odd/even positioning

Automatic Positioning Rules:

  • :nth-child(odd) → Left column (grid-column: 1)
  • :nth-child(even) → Right column (grid-column: 2)
  • Progressive top offset: calc(-2N * padding)
  • Scalable: just add more boxes!

Tailwind Odd/Even Classes:

  • odd:sm:border-r-4 (left boxes, right border)
  • even:sm:border-l-4 (right boxes, left border)
  • odd:sm:rounded-tr-3xl (left box radius)
  • even:sm:rounded-tl-3xl (right box radius)

Key Features:

  • Exact same overlapping pattern as 5.2 version
  • Content-driven height with min-height fallback
  • Responsive: stacks vertically on mobile with positioning reset
  • Custom CSS for precise top positioning using --bento-padding variable
