Using Google Web Fonts

  • There are privacy concerns with using Google Web Fonts, and sites with high privacy needs may want to avoid the service 1
  • Self-hosting fonts is an alternative that avoids sending any data to Google 2
  • Some fonts available through Google Fonts can be downloaded and self-hosted, while others may have licenses that don't allow redistribution 3
  • Google can count users and track information like IP addresses and browser versions of visitors to sites using Google Fonts 4.
