CSS Turn Unit Examples

1. Basic Single Turn Rotation

A simple square that completes exactly one full rotation (1 turn = 360°) continuously.

@keyframes basicRotate { from { transform: rotate(0turn); } to { transform: rotate(1turn); } } .rotating-square { animation: basicRotate 3s linear infinite; }

2. Multi-Turn Rotation with Scaling

A gear-like element that rotates 5 full turns while scaling up and down, demonstrating fractional and multiple turn values.

@keyframes multiTurnSpin { 0% { transform: rotate(0turn) scale(1); } 50% { transform: rotate(2.5turn) scale(1.2); } 100% { transform: rotate(5turn) scale(1); } } .spinning-gear { animation: multiTurnSpin 4s ease-in-out infinite; }

3. Clock Animation with Different Turn Speeds

A functional clock where the minute hand completes 1 turn per second, and the hour hand completes 1 turn in 12 seconds, using fractional turn values for precise positioning.

/* Hour hand starts at 3 o'clock position */ @keyframes hourHand { from { transform: translateX(-50%) rotate(0.25turn); } to { transform: translateX(-50%) rotate(1.25turn); } } /* Minute hand completes full rotation */ @keyframes minuteHand { from { transform: translateX(-50%) rotate(0turn); } to { transform: translateX(-50%) rotate(1turn); } }

About the Turn Unit

The turn unit represents complete rotations where:
• 1 turn = 360 degrees = 2π radians = 400 gradians
• 0.5 turn = 180 degrees (half rotation)
• 0.25 turn = 90 degrees (quarter rotation)
• 2 turn = 720 degrees (two full rotations)
