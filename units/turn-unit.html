Technical CSS Turn Unit Examples

1. Animation Property with Turn Values


@keyframes rotate90 {
    to { transform: rotate(0.25turn); }  /* 90 degrees */
}

@keyframes rotate180 {
    to { transform: rotate(0.5turn); }   /* 180 degrees */
}

@keyframes rotate360 {
    to { transform: rotate(1turn); }     /* 360 degrees */
}
Technical use: Turn units in animations provide exact rotation control. 0.25turn = 90°, 0.5turn = 180°, 1turn = 360°.

2. Conic Gradient Color Stops with Turn Units


/* Quarter split */
background: conic-gradient(from 0turn, red 0.25turn, blue 0.25turn);

/* Three equal sections */
background: conic-gradient(from 0turn,
    red 0.33turn,
    green 0.33turn 0.66turn,
    blue 0.66turn);

/* Precise positioning with offset start */
background: conic-gradient(from 0.125turn,
    #000 0.25turn,
    #fff 0.25turn 0.75turn,
    #000 0.75turn);
Technical use: Turn units in conic gradients allow precise angular color stops. More intuitive than degrees for circular divisions.

3. Transform Functions with Turn Precision


/* Static rotation */
.element { transform: rotate(0.125turn); }  /* 45 degrees */

/* Combined transforms */
.element { transform: rotate(0.25turn) scale(1.2) translateX(20px); }

/* 3D rotations */
.element { transform: rotateX(0.25turn) rotateY(0.125turn); }
Technical use: Turn units work with all rotation transform functions: rotate(), rotateX(), rotateY(), rotateZ(), rotate3d().

4. CSS Custom Properties (Variables) with Turn Values

(hover to see change) 

.element {
    --rotation: 0.1turn;
    --color-stop: 0.3turn;

    background: conic-gradient(from var(--rotation),
        red var(--color-stop),
        blue var(--color-stop));
    transform: rotate(var(--rotation));
    transition: transform 0.3s ease;
}

.element:hover {
    --rotation: 0.5turn;
}
Technical use: Turn values can be stored in CSS variables for dynamic theming and consistent rotation values across components.

5. Filter Hue-Rotate with Turn Units


.original { filter: hue-rotate(0turn); }      /* No change */
.quarter  { filter: hue-rotate(0.25turn); }   /* 90° hue shift */
.half     { filter: hue-rotate(0.5turn); }    /* 180° hue shift */
.full     { filter: hue-rotate(1turn); }      /* 360° = back to original */
Technical use: Hue-rotate filter accepts turn units for color wheel rotations. Useful for theme variations and color accessibility adjustments.

Turn Unit Technical Reference


/* Conversion Reference */
0turn     = 0deg      = 0rad       = 0grad
0.25turn  = 90deg     = π/2 rad    = 100grad
0.5turn   = 180deg    = π rad      = 200grad
0.75turn  = 270deg    = 3π/2 rad   = 300grad
1turn     = 360deg    = 2π rad     = 400grad

/* Supported Properties */
- transform: rotate(), rotateX(), rotateY(), rotateZ(), rotate3d()
- background: conic-gradient()
- filter: hue-rotate()
- Custom properties (CSS variables)
- Animation keyframes
